# Kit::Support

A toolkit of support libraries and Ruby core extensions.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'kit-support'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install kit-support
```
