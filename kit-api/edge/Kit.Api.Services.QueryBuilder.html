    QueryBuilder module
    Kit::Api::Services::QueryBuilder
    View Source

    Build a Query (a fully actionable AST) from a Request.

    Each level is a query node, even when there are 1:N or N:N scenarios. For instance Author -> Books -> Chapters will be resolved as 3 corresponding query nodes total, eventhough there are many books, with many chapters.

    Each type of relationship generates a query node even if they target the same resource. For instance Author -> [Chapter | FirstChapter] will generate 3 corresponding query nodes total, eventhough the two relationships Chapter and FirstChapter target the same Resource (Chapter)
