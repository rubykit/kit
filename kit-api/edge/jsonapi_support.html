    JsonApi support

    Kit::Api::JsonApi helps you to implement a server side API conforming to JSON:API v1.1 spec.

    The goals of this gems are to:

    • describe endpoints that can serve & modify JSON:API Documents
    • parse & verify JSON:API query strings requests
    • describe Resources that can be exposed as JSON:API Documents

    Support

    Current status:

    • [x] JSONAPI Document encoding
    • [x] Links
    • [x] Sparse fieldsets Note that fieldsets are per type and not per relationship.
    • [x] Sorting
    • [x] Pagination
    • [x] Relationships: links (including nested pagination)
    • [x] Relationships: includes
    • [x] Relationships: compound documents (sideloading)
    • [x] Top level resources queries
    • [ ] Top level relationships queries
    • [ ] Top level meta data
    • [ ] Errors

    Not supported:

    • Polymorphic relationships. Nothing in the spec prevents them, but this seems mostly like an anti-pattern: trying to hide a join model works only if there are no attributes in use on that model, appart from foreign keys. This is often unlikely.
